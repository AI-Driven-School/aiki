# Marketing Materials

This directory contains promotional content for claude-codex-collab. These files are **not tracked in git** to keep the repository focused on the tool itself.

## Files

| File | Platform | Description |
|------|----------|-------------|
| `TWITTER_THREAD.md` | X (Twitter) | Thread template for announcement |
| `REDDIT_POSTS.md` | Reddit | Post templates for relevant subreddits |
| `ZENN_ARTICLE.md` | Zenn | Japanese technical article |

## Usage

1. Copy the template content
2. Customize for your audience
3. Post to the respective platform

These files are excluded from git via `.gitignore`. To share with collaborators, use a separate channel (Google Drive, Notion, etc.).
